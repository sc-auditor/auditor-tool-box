Smart Contract Audit Tools

Here’s a list of tools I’ll be using for smart contract auditing:

  • Slither - Static analysis tool for Solidity.
  • Mythril - Security analysis tool for Ethereum smart contracts.
  • Echidna - Property-based fuzzer for Ethereum.
  • Remix IDE - Development environment for Solidity.
  • Hardhat - Ethereum development framework.
